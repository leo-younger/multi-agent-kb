API 接口文档 v1.0

一、用户相关接口
1. 用户查询接口 GET /api/user/query
   功能：根据用户 ID 查询用户信息
   负责人：张三
   依赖：认证模块、用户系统

2. 用户登录接口 POST /api/user/login
   功能：用户身份验证，返回 Token
   负责人：张三
   依赖：认证模块

二、订单相关接口
3. 订单创建接口 POST /api/order/create
   功能：创建新订单
   负责人：王五
   依赖：订单系统、库存系统、支付系统

4. 订单查询接口 GET /api/order/query
   功能：查询订单列表和详情
   负责人：王五
   依赖：订单系统

三、文档相关接口
5. 文件上传接口 POST /api/document/upload
   功能：上传文档到知识库
   负责人：李四
   依赖：知识库模块、解析服务

6. 文档搜索接口 GET /api/document/search
   功能：全文检索知识库文档
   负责人：张三
   依赖：知识库模块

四、数据相关接口
7. 数据同步接口 POST /api/data/sync
   功能：定时同步外部数据源
   负责人：数据部
   依赖：调度模块

五、系统说明
所有接口通过网关模块统一暴露，支持 HTTPS 协议。
认证模块负责接口的鉴权和限流。
日志系统记录所有接口的调用日志。
